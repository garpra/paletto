# Paletto

> A collection of elegant color themes specially curated for Visual Studio Code.

## Available Themes

| Theme | Style |
|-------|-------|
| Paletto — Tokyo Night Dark | Dark |
| Paletto — Gruvbox Dark | Dark |

## Activate a Theme

Go to:
```
File > Preferences > Color Theme
```
Then search for any theme starting with **Paletto**.
